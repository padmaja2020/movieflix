Movie Flix Documentaion

The web application will provide users with access to information about different movies, directors, and genres.
Users will be able to sign up, update their personal information, and create a list of their favorite movies

Request URL HTTP Method Request body data format Response body data format
List of all movies /movies GET none JSON object holding data about all the movies
Get data about single movie by title /movies[name] GET none JSON object holding data about the movie containing a description, genre, director, image URL, whether it's featured or not. Example:

{ description : "The Devil Wears Prada is a 2006 American comedy-drama film", genre: "comedy", director: "David Frankel", image URL:"", featured:true }

Get data about a genre by name/title /movies/[title]/genre GET none JSON object holding data about the genre.
Get data about a director by name /movies/director/[name] GET none JSON object holding data about the director containing bio, birth year, death year. Example:

{ name: "David Frankel", bio:"David Frankel is an American film director, screenwriter and producer", birth year:"April 2, 1959", death year: "" }

Allow new users to register /users POST A JSON object holding data about the user Example:

{ username : "", password:"", email:"", date of birth:"", favorites:"" }

A JSON object holding data about the user that was added, including an ID: Example:

{ id:"", username : "", password:"", email:"", date of birth:"" }
